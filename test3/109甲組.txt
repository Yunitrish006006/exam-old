3. 
Solve the differential equations, where x(0) = 1 and y(0) = 0.
    dx/dt = 3x-4y, dy/dt = 2x-3y

    dx / (3x - 4y) = dt => 積分 => ∫(1 / (3x - 4y)) dx = ∫dt
    1 / (3x - 4y) = A / 3x + B / (-4y) => 同乘 => 1 = A(-4y) + B(3x)

    因為 x(0) = 1 和 y(0) = 0
    1 = A(0) + B(3*1)   =>  B = 1/3
    1 = A(-40) + (1/3)(31)  =>  A = 1/3
    1 / (3x - 4y) = (1/3) / 3x + (1/3) / (-4y) => 積分 => (1/3) ∫ (1 / 3x) dx - (1/3) ∫ (1 / 4y) dy = ∫ dt
    
    (1/3) ln|3x| - (1/3) ln|4y| = t + C
    ln|3x / 4y| = 3t + 3C
    3x / 4y = e^(3t + 3C)  # 另 e^(3C) = K
    3x / 4y = Ke^(3t)
    --------------------------------------------
    dy/dt = 2x - 3y
    dy / (2x - 3y) = dt
    ln|2x / 3y| = 2t + D
    2x / 3y = Fe^(2t)
    y = (2x / 3F)e^(2t)
    1 = (40K)e^(30)

    Answer:
    x = (4yK)e^(3t)
    y = 0
    & K = 任何一個不是0的常數
